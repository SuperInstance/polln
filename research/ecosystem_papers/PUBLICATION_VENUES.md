- **Short papers**: 15-20 pages

## Submission Requirements Checklist

### For Each Submission

- [ ] Paper formatted according to venue template
- [ ] Page limit compliance
- [ ] Anonymous version prepared (if required)
- [ ] Supplementary materials prepared
- [ ] Code repository cleaned and documented
- [ ] Data license confirmed (CC-BY-4.0 recommended)
- [ ] Author information confirmed
- [ ] Conflict of interest disclosed
- [ ] Open review preference set
- [ ] Presentation slides prepared (if accepted)

### Venue-Specific Requirements

**AAAI**:
- Use AAAI template
- Anonymous submission required
- Supplementary materials allowed (up to 100MB)

**ICML**:
- Use ICML 2026 template
- Anonymous submission required
- Code review available (optional)

**NeurIPS**:
- Use NeurIPS template
- Anonymous submission required
- Supp materials (PDF, up to 100MB)

**KDD**:
- Use ACM template
- Non-anonymous submission
- Reproducibility checklist required
