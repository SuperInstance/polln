## Mathematical Integration

### Pressure Differential (from P25)
```
ΔP_i = P_in - Σ_j (P_j / R_ij)
where P_j = downstream pressure, R_ij = flow resistance
```

### Transfer Entropy (from P27)
```
T(P_j → P_i) = H(P_{i,t+1}|P_{i,t}) - H(P_{i,t+1}|P_{i,t}, P_{j,t})
```

### Emergence Condition
```
Emergence if:
  1. T(P_j → P_i) > threshold_entropy
  2. d(ΔP_i)/dt > critical_rate
  3. Shannon diversity(strategies) > baseline
```

```python
# Both P25 and P27 use transfer entropy
def transfer_entropy(signal_source, signal_target, bins=10):
    """T(source → target) = H(target_next|target) - H(target_next|target, source)"""
    # Discretize signals
    source_disc = np.digitize(signal_source, bins)
    target_disc = np.digitize(signal_target, bins)

    # Compute conditional entropies
    h_target_given_target = conditional_entropy(target_disc[1:], target_disc[:-1])
    h_target_given_both = conditional_entropy(
        target_disc[1:],
        zip(target_disc[:-1], source_disc[:-1])
    )

    return h_target_given_target - h_target_given_both
```
