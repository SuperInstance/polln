## ROI Calculator

### Strategic Tier ROI Calculator

```python
def calculate_strategic_roi(
    company_size: str,
    engineering_cost: float,
    time_to_market_months: float,
    infrastructure_budget: float,
    compliance_required: bool
):
    """
    Calculate ROI for Strategic tier partnership

    Args:
        company_size: 'startup', 'mid-sized', 'enterprise'
        engineering_cost: Annual engineering cost ($)
        time_to_market_months: Months to launch without SuperInstance
        infrastructure_budget: Annual infrastructure budget ($)
        compliance_required: Whether SOC2/GDPR compliance needed

    Returns:
        ROI analysis with savings and payback period
    """

    # Annual partnership cost
    partnership_cost = 120000

    # Savings calculations
    engineering_savings = engineering_cost * 0.4  # 40% faster development
    infrastructure_savings = infrastructure_budget * 0.6  # 60% reduction
    time_to_market_savings = (time_to_market_months / 12) * \
                             (engineering_cost / 2)  # Half team idle
    compliance_savings = 150000 if compliance_required else 0

    total_savings = (engineering_savings +
                    infrastructure_savings +
                    time_to_market_savings +
                    compliance_savings)

    roi = ((total_savings - partnership_cost) / partnership_cost) * 100
    payback_months = partnership_cost / (total_savings / 12)

    return {
        'annual_savings': total_savings,
        'partnership_cost': partnership_cost,
        'net_benefit': total_savings - partnership_cost,
        'roi_percentage': roi,
        'payback_period_months': payback_months
    }
```

**Example Usage**:
```python

    company_size='mid-sized',
    engineering_cost=2000000,  # $2M/year engineering team
    time_to_market_months=12,  # 12 months to launch
    infrastructure_budget=500000,  # $500K infrastructure
    compliance_required=True
)

# Result:

# - Payback Period: 0.95 months
```

### Enterprise Tier ROI Calculator

```python
def calculate_enterprise_roi(
    annual_revenue: float,
    infrastructure_budget: float,
    compliance_budget: float,
    engineering_team_size: int,
    uptime_requirement: float,
    industry: str
):
    """
    Calculate ROI for Enterprise tier partnership

    Args:
        annual_revenue: Company annual revenue ($)
        infrastructure_budget: Annual infrastructure budget ($)
        compliance_budget: Annual compliance and audit budget ($)
        engineering_team_size: Number of engineers
        uptime_requirement: Required uptime (e.g., 0.9999 for 99.99%)
        industry: Industry sector

    Returns:
        Comprehensive ROI analysis
    """

    # Annual partnership cost
    partnership_cost = 600000

    # Direct cost savings
    infrastructure_savings = infrastructure_budget * 0.7
    compliance_savings = compliance_budget * 0.8
    support_savings = engineering_team_size * 150000 * 0.3  # 30% productivity gain

    # Risk reduction (avoided incidents)
    if uptime_requirement >= 0.9999:
        incident_savings = annual_revenue * 0.001  # Avoid 0.1% downtime impact
    else:
        incident_savings = annual_revenue * 0.0005

    # Time to market value
    engineering_savings = engineering_team_size * 200000 * 0.5  # 50% faster

    total_savings = (infrastructure_savings +
                    compliance_savings +
                    support_savings +
                    incident_savings +
                    engineering_savings)

    roi = ((total_savings - partnership_cost) / partnership_cost) * 100
    payback_months = partnership_cost / (total_savings / 12)

    return {
        'annual_savings': total_savings,
        'partnership_cost': partnership_cost,
        'net_benefit': total_savings - partnership_cost,
        'roi_percentage': roi,
        'payback_period_months': payback_months,
        'breakdown': {
            'infrastructure': infrastructure_savings,
            'compliance': compliance_savings,
            'support_productivity': support_savings,
            'risk_avoidance': incident_savings,
            'engineering_velocity': engineering_savings
        }
    }
```
