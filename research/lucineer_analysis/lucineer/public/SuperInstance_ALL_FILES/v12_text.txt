PART 2: CORRECTED TECHNICAL ANALYSIS
2.1 Memory Architecture: The Real Bottleneck
For autoregressive LLM decode, KV cache bandwidth dominates, not weight fetch. Weights are accessed once per layer; KV cache is
accessed every token. This changes the architecture focus from weight storage to KV cache optimization.
                              Context                 KV Cache               Bandwidth @25 tok/s                  LPDDR4 Capable?
