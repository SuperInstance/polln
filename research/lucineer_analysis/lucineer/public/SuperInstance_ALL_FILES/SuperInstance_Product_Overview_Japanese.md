### 3.2 C₄群重み量子化技術

#### 3.2.1 数学的基盤

iFairyアーキテクチャ（北京大学、arXiv:2508.05571）は、以下の重み集合を使用します：

$$W = \{+1, -1, +i, -i\}$$

これは乗法に関する**巡回群C₄**を形成し、ハードウェア実装において大きな利点を持ちます。

| 重み (w) | 回転角 | 演算内容 | ハードウェアコスト |
|----------|--------|----------|-------------------|
| +1 | 0° | 恒等変換 | 0ゲート |
| -1 | 180° | 両成分の反転 | NOTゲート2個 |
| +i | +90° | 実部・虚部交換、実部反転 | 配線クロス + NOT 1個 |
| -i | -90° | 実部・虚部交換、虚部反転 | 配線クロス + NOT 1個 |

**乗算排除定理**: 任意の重み w ∈ W と複素活性化 z = a + bi に対し、積 w × z は**算術乗算を一切必要としない**。

#### 3.2.2 回転累積ユニット（RAU）仕様

| 項目 | 仕様 |
|------|------|
| ゲート数 | 約150ゲート/RAU |
| レイテンシ | 1サイクル（組み合わせパス） |
| 消費エネルギー | 約0.15 pJ/演算（28nmプロセス） |
| 面積 | 20-50 μm² |

### 3.3 実装方式比較

| 実装方式 | ゲート数 | 消費電力 (pJ/演算) | レイテンシ (サイクル) | 面積 (μm²) |
|----------|----------|-------------------|---------------------|------------|
| FP16乗算器 | 3,000-5,000 | 50-100 | 3-5 | 2,000-3,000 |

### 5.1 API仕様

#### 5.1.1 REST API

```python
# 基本的な推論リクエスト
import requests

response = requests.post(
    "http://localhost:8080/v1/completions",
    json={
        "prompt": "こんにちは、",
        "max_tokens": 100,
        "temperature": 0.7
    }
)

result = response.json()
print(result["text"])
```

#### 5.1.2 Python SDK

```python
from superinstance import SuperInstance

# デバイス初期化
device = SuperInstance()

# 推論実行
response = device.generate(
    prompt="AIの将来について教えてください",
    max_tokens=200,
    temperature=0.8
)

print(response.text)

# ストリーミング出力
for token in device.stream(prompt="長い文章を生成して"):
    print(token, end="", flush=True)
```
