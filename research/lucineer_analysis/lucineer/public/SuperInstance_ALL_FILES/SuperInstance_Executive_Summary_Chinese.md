# 二、核心技术

## 2.1 专用架构

我们的核心技术突破在于将神经网络权重直接编码到硅片的金属层中，从根本上消除了传统AI芯片的内存瓶颈。

### 技术优势对比

| 技术指标 | 传统GPU方案 | SuperInstance方案 | 提升倍数 |
|----------|------------|-------------------|----------|
| **权重读取功耗** | 50-100 pJ/bit | **~0 pJ/bit** | 无限 |
| **内存带宽瓶颈** | 严重 | **不存在** | - |
| **推理效率** | 1.7 tok/s/W | **50 tok/s/W** | **29倍** |
| **芯片面积利用率** | 60-70% | **85%+** | 1.2倍 |

### iFairy C₄ 量化架构

基于北京大学研究成果（arXiv:2508.05571），我们采用第四单位根权重量化：

$$W = \{+1, -1, +i, -i\}$$

**核心突破**：用简单的数据旋转和符号反转替代复杂的乘法运算。

| 运算类型 | 传统FP16乘法器 | iFairy旋转累加单元 |
|----------|---------------|-------------------|
| 门电路数量 | 3,000-5,000 | **~150** |
| 每次运算功耗 | 50-100 pJ | **0.1-0.5 pJ** |
| 延迟周期 | 3-5 | **1** |
| 芯片面积 | 2,000-3,000 μm² | **20-50 μm²** |

**结果**：每瓦性能提升100-500倍。

## 2.2 三层存储架构

```
┌─────────────────────────────────────────────────────────────┐
│                    存储层次架构                               │
├─────────────────────────────────────────────────────────────┤
│  第0层：专用架构权重（只读）                                  │
│  ├── 存储：20亿参数 × 2比特 = 500 MB                        │
│  ├── 编码：金属层布线（零访问时间）                          │
│  ├── 带宽：无限（硬连线至计算单元）                          │
│  └── 功耗：零（无需读取操作）                                │
├─────────────────────────────────────────────────────────────┤
│  第1层：片上SRAM（KV缓存 + 激活值）                          │
│  ├── 容量：21-42 MB                                        │
│  ├── 技术：28nm 6T SRAM                                    │
│  └── 访问时间：1-2周期                                      │
├─────────────────────────────────────────────────────────────┤
│  第2层：LPDDR5（扩展上下文）                                 │
│  ├── 容量：512 MB - 2 GB（可配置）                          │
│  └── 带宽：51.2 GB/s                                       │
└─────────────────────────────────────────────────────────────┘
```
