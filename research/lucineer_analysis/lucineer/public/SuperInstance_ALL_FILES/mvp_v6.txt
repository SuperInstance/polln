TIER 1: CORRECTED COST MODEL

2.1 The SRAM Cost Reality
v5.0 claimed "512MB SRAM for $4." This was a fatal error. At 28nm process node:


                    Memory Type                Area per MB                Cost per MB        256MB Total

                  On-die SRAM (6T)              ~0.84 mm2                   $0.08-0.12          $20-30

                  External LPDDR4             N/A (package)                 $0.005-0.01          $1-3

                    External DDR4             N/A (package)              $0.003-0.006         $0.75-1.50

                                          Table 3: Memory Cost Comparison at 28nm

Solution: Use external LPDDR4 (512MB) for KV cache and activations. Weights remain mask-locked in metal
layers (zero SRAM cost). This adds ~50-100ns latency per access but maintains acceptable tok/s with pipelined
architecture.
2.2 Power Budget Validation
v5.0 claimed "5W from Pi 5V rail." Verified research shows:


                 Parameter                 v5.0 Claim                     Verified Reality                  Source

            5V pin max current           1A (implied)                 1-1.5A (copper limited)             Pi Forums

              Sustained power           5W continuous              3-4W before thermal throttle        Thermal analysis

                Peak power                    N/A                    5-7.5W for <30 seconds           Power supply spec

                                               Table 4: Power Budget Reality Check

Design implication: Target 3W sustained operation with burst capability to 5W. This requires thermal throttling API
and performance mode switch in software.
