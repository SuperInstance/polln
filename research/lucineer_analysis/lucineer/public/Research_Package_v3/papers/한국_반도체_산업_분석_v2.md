본 보고서는 삼진 가중치(Ternary Weight) 추론 칩의 한국 시장 진입 전략과 반도체 산업 생태계 분석을 다룹니다. 한국은 세계 최대 메모리 반도체 생산국으로서 전략적 파트너십 기회를 제공합니다.

## 제1장: 한국 반도체 산업 개요

### 1.1 산업 구조

한국 반도체 산업은 메모리 중심의 독특한 구조를 가집니다:

**주요 기업 포지셔닝**:

| 기업 | 분야 | 매출 (2025) | 세계 점유율 |
|------|------|-------------|-------------|
| 삼성전자 | 메모리/파운드리 | $75B | DRAM 44%, NAND 35% |
| SK하이닉스 | 메모리 | $32B | DRAM 28% |
| DB하이텍 | 파운드리 | $1.2B | 특수공정 8" |
| 키이엠 | 후공정 | $0.8B | 패키징 전문 |

### 1.2 파운드리 생태계

**삼성전자 파운드리 라인업**:

```
공정 노드    용량         특징
───────────────────────────────────
3nm GAA      파일럿      최신 기술
5nm          양산        고성능
7nm          양산        밸런스
14nm         양산        메인스트림
28nm         대용량      성숙/저비용 ← 타겟
45nm         대용량      저비용
```

**28nm HPM 공정 상세**:
- 웨이퍼 가격: $3,500
- 수율 (Yield): ~85%
- 리드타임: 8주
- 적용 분야: IoT, Automotive, Consumer

### 1.3 패키징 생태계

**주요 OSAT (후공정) 기업**:

| 기업 | 전문 분야 | 월 생산능력 | 가격 경쟁력 |
|------|-----------|-------------|-------------|
| 앰코테크놀로지 | BGA, QFN | 50M units | 중간 |

## 제4장: 제조 전략

### 4.1 웨이퍼 공급 계약

**필요 용량 산정**:

```
Year 1:  5K 칩 × 1.2 (수율) = 6K 다이
         6K / 800 (다이/웨이퍼) = 8 웨이퍼/월
         
Year 2:  30K 칩 = 38 웨이퍼/월

Year 3:  100K 칩 = 125 웨이퍼/월
```

**계약 조건 제안**:

| 항목 | 조건 |
|------|------|
| 최소 주문 | 50 웨이퍼/LOT |
| 단가 | $3,200/웨이퍼 |
| 납기 | 8주 |
| 수율 보증 | ≥ 80% |

### 4.2 패키징 아웃소싱

**추천 OSAT**:

1순위: **앰코테크놀로지** (한국)
- BGA-256 전문
- 가격: $0.80/칩
- 품질: 높음

2순위: **JCET** (중국)
- 가격: $0.50/칩
- 품질: 중간
- 리스크: 지정학적

### 4.3 테스트 전략

**테스트 플로우**:

```
┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐
│ Wafer   │───►│ FT      │───►│ Burn-In │───►│ Final   │
│ Sort    │    │ Test    │    │ (24h)   │    │ Test    │
└─────────┘    └─────────┘    └─────────┘    └─────────┘
    │              │              │              │
    ▼              ▼              ▼              ▼
  수율 85%      기능 테스트    신뢰성 검증   속도/전력
  불량 제거     95% 패스       99% 패스      98% 패스
```

**테스트 비용**:
- Wafer Sort: $0.10/다이
- FT: $0.15/칩
- Burn-In: $0.20/칩
- Final: $0.10/칩
- **Total: $0.55/칩**
