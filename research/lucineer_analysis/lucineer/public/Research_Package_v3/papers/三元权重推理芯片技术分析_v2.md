### 1.2 三元权重的信息论基础

三元权重（取值{-1, 0, +1}）从信息论角度具有独特优势：

**香农熵计算**：
```
H(X) = -Σ p(x) log₂ p(x)

均衡三元分布：
H(X) = -3 × (1/3) × log₂(1/3) = 1.585 bits
```

相比INT8的8 bits表示，实现**5.06倍压缩**。

## 第二章：核心技术架构

### 2.1 掩码锁定技术

掩码锁定通过在制造时将权重固化到ROM：

**技术优势**：
- 面积：减少85%
- 功耗：减少70%
- 安全：物理级IP保护

### 2.2 简化算术单元（RAU）

三元乘法大幅简化电路：

```
w × a 运算：
  w = +1：输出 = a（直通）
  w = 0： 输出 = 0（断开）
  w = -1：输出 = -a（取反）

面积减少：85%
功耗减少：70%
延迟减少：50%
```
