## Task Breakdown by Agent

### Agent 1: Market Research Agent

#### Task 1.1: Customer Database Creation
**Priority:** 🔴 CRITICAL
**Duration:** 3 days
**Dependencies:** None

**Description:**
Build comprehensive database of potential customers.

**Steps:**
1. Research edge AI companies
2. Identify key decision makers
3. Gather contact information
4. Score leads by priority
5. Create CRM structure

**Outputs:**
- Customer database (200+ contacts)
- Lead scoring matrix
- CRM setup
- Outreach sequence templates

**Validation:**
- Database has all required fields
- Lead scores are actionable
- CRM is functional

---

#### Task 1.2: Interview Script Development
**Priority:** 🔴 CRITICAL
**Duration:** 2 days
**Dependencies:** None

**Description:**
Create structured interview scripts for customer discovery.

**Inputs:**
- Customer segment definitions
- Key hypotheses to test
- Product value proposition

**Steps:**
1. Define hypotheses to test
2. Create question framework
3. Develop segment-specific scripts
4. Create scoring rubric
5. Pilot test internally

**Test Hypotheses:**
1. Customers need low-power inference (<5W)
2. Customers value on-device privacy
3. Customers prefer fixed-function over programmable
4. Price sensitivity at $45-65 range
5. Design-in cycle is 6-12 months

**Outputs:**
- Interview scripts (3 variants)
- Hypothesis testing framework
- Scoring rubric
- Recording/logging system

**Validation:**
- Scripts test all hypotheses
- Scoring is objective
- Takes <30 minutes per interview
