## Visual Language Specifications

### Color Palette
```
Primary Colors:
- #22c55e (Green 500) - Primary accent, success, chip/silicon
- #14b8a6 (Teal 500) - Secondary accent, neural, AI
- #8b5cf6 (Purple 500) - Tertiary accent, creativity, learning

Background Colors:
- #0f0f0f (Near black) - Primary background
- #1a1a1a (Dark gray) - Secondary background
- #0a0a0a (Deepest) - Contrast sections

Accent Colors:
- #f59e0b (Amber 500) - Energy, power, warnings
- #ec4899 (Pink 500) - Highlights, creativity
- #3b82f6 (Blue 500) - Links, trust
```

### Typography Scale
```
Headlines: Bold, tight tracking
- H1: 5xl-8xl, font-bold
- H2: 3xl-4xl, font-bold  
- H3: xl-2xl, font-semibold

Body: Readable, slightly lighter
- Large: lg, leading-relaxed
- Normal: base
- Small: sm

Code/Mono: Technical elements
- font-mono, text-sm
```

### Shape Language
```
Corners: rounded-xl (12px) to rounded-2xl (16px)
Shadows: shadow-xl for emphasis, shadow-lg for cards
Borders: border-dark-600 as default, colored borders for emphasis
Glows: blur-xl opacity-50 for interactive elements
```

### Animation Guidelines
```
Transitions: transition-all duration-300
Hover: scale(1.02), brightness(1.1)
Pulse: animate-pulse for attention
Fade: opacity transitions for modals
```
