# TernaryAir Complete Project v2.0

## Project Overview

**TernaryAir** is an offline-first educational platform teaching AI and chip design concepts through timing games and interactive simulations. From playground games to professional FPGA design — one platform, infinite depth.

---

## Quick Start

```bash
cd web
bun install
bun run dev
```

Open http://localhost:3000

## Key Features

### Offline-First
- Works immediately on page load
- No signup required
- Progress saved locally (IndexedDB)
- Optional Cloudflare sync

### Multi-Age Learning
| Age | Focus | Activities |
|-----|-------|------------|
| 4-10 | Play-based | Traffic lights, dominoes |
| 11-14 | Bridge | Animation, beat programming |
| 15-18 | Technical | Digital logic, timing |
| 18+ | Professional | FPGA, CDC, pipelines |

### API Integration
- **Free**: Groq, x.ai, HuggingFace
- **Low Cost**: DeepSeek, z.ai, Kimi, Mini
- **Major**: OpenAI, Anthropic, Google, Mistral
- **Local**: Ollama, LM Studio
