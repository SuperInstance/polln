# POLLN Research Archive

## Overview

This directory contains archived research documents, simulation results, and historical development records from the POLLN project. All research has been preserved while organizing it into a clean, accessible structure.

## Archive Structure

### `/archive/research-breakdown/`
**59 files** - Breakdown Engine Research (R2-R8)

Contains comprehensive research on:
- **R2**: Fractured AI Boxes, Tokenization, Transformer Layers, Model Cascade, Ambiguity Resolution, Orchestrator Protocol
- **R3**: AgentCell Integration, Box Communication, Learning, Testing, Visualization, Runtime Engine
- **R4**: Creativity Engine, Morphogenesis, Quantum Boxes, Self-Awareness, Semantic Memory, Swarm Intelligence, Temporal Dynamics
- **R5**: Box Aesthetics, Culture, Dreams, Emotion, Language, Philosophy, Consciousness
- **R6**: Death, Immortality, Love, Mythopoesis, Society, Spirituality, Utopia
- **R7**: Ascension, Cosmos, Dimensionality, Omega Point, Omniscience, Synthesis, Time Travel
- **R8**: Apophatic, Eternal Return, Meta-Cosmology, Novel Physics, Omega Singularity, Post-Metaphysics, Transcendental Math

**Key Documents:**
- `BREAKDOWN_R*_SUMMARY.md` - Executive summaries for each round
- `BREAKDOWN_R*_QUICKREF.md` - Quick reference guides

### `/archive/wave-reports/`
**15 files** - Spreadsheet Development Waves

Contains reports from Waves 4-7:
- **WAVE4**: UI Components Foundation
- **WAVE5**: Creative Insights, Implementation Specs, Visual Guide
- **WAVE6**: Architecture Diagrams, Backend Summary, Deliverables, Implementation Guide
- **WAVE7**: Checklist, Diagrams, Quickstart, Research Summary

### `/archive/simulation-results/`
**18 files** - Simulation Scripts and Results

Python simulation scripts and output files:
- `kimi_*` - Kimi creative vision simulations
- `learning_distillation_simulations.py` - Learning and distillation research
- `run_simulations.py`, `run_sensation_sims.py` - Main simulation runners
- `sim_logic_costs.py` - Logic cost analysis
- `simulate_network_scaling.py` - Network scaling experiments
- `wave6_backend_simulations.py` - Backend performance simulations
- Associated JSON output files and logs

### `/archive/agent-reports/`
**25 files** - Agent Onboarding and Roadmaps

Historical agent coordination documents:
- Alpha through Eta agent onboarding and roadmaps
- Milestone reports and summaries
- Planning team documentation

### `/archive/test-outputs/`
**102 directories** - LoRA Training Outputs

Archived test outputs from LoRA training experiments:
- Matrix files and metadata for training runs
- Organized by experiment UUID
